# Bordo — Website For Online Restaurant Menus 🍽️

Bordo is a modern, responsive web application designed for restaurant digital menus. It allows customers to view dishes, prices, and descriptions easily on any device.

## 🎥 Video Demonstration
I have recorded a detailed walkthrough showing how the site looks on both Desktop and Mobile devices, along with its main features.

* [Watch the Mobile Demo on YouTube](https://youtu.be/9XHa7sMlVEI)
* [Link to the Site](https://nikita-kolenchenko.github.io/Bordo/)

## ✨ Key Features
* **Fully Responsive Design:** Optimized for a seamless experience on smartphones, tablets, and desktops.
* **Modern UI/UX:** A clean and elegant interface that matches a restaurant's premium feel.
* **Interactive Menu:** Easy-to-navigate categories and dish details.
* **Fast Performance:** Lightweight and optimized for quick loading.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript.
* **Design:** Focused on "Mobile First" philosophy.

## 📱 Mobile vs Desktop
You can see the difference in the layout in my videos. The project uses media queries to ensure the menu is readable and clickable even on small screens.
